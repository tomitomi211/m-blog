Rust製のTauriを調べる & ハロワまで

Tauriの概要
- TauriはRustベースのデスクトップアプリケーション開発フレームワーク
- 軽量でセキュアなクロスプラットフォームのデスクトップアプリ開発に向いている

特徴
- 非常に小さなバイナリサイズで実現可能
- Electoronに比べて大きなランタイムを含まない
- クロスプラットフォーム : win, mac, linuxで動作する
- フロントエンドは任意のwebフレームワークを使用できる

公式サイト
- https://tauri.app/



TauriのHello worldまでの手順
①ひとまずRustを入れるところから
Rustを入れる
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
. "$HOME/.cargo/env" (上記を実行したらこのパスも通せと言われる)

Rustとcargoが入ったか確認
rustc --version
cargo --version
->
rustc 1.84.0 (9fc6b4312 2025-01-07)
cargo 1.84.0 (66221abde 2024-11-19)
で入ってるのを確認した。

Tauriのコマンドを入れる
cargo install tauri-cli
echo 'export PATH="$HOME/.cargo/bin:$PATH"' >> ~/.zshrc

Viteを入れる
npm install vite --save-dev

②やっとTauriを入れる
npm create tauri-app
を実行してyesを連打

npm run tauri dev 
これでGUIを起動
