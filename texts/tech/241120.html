Tone.jsを使って簡単なサンプルを作成します。

Tone.jsはWeb Audio APIを簡単に操作できるライブラリで、音楽や音の合成を行うために非常に便利です

以下のコードは、Tone.jsを使ってシンセサイザーで音を鳴らす簡単な例です。

1. HTMLファイルを作成

まず、HTMLファイルを作成して、Tone.jsライブラリを読み込みます。CDNを使って簡単にTone.jsをインクルードできます。 

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Tone.js Simple Example</title>
                <script src="https://cdn.jsdelivr.net/npm/tone@14.8.28/build/Tone.js"></script>
            </head>
            <body>
                <h1>Tone.js Simple Example</h1>
                <button id="playButton">Play Sound</button>
            
                <script>
                    // Tone.jsのシンセサイザーを作成
                    const synth = new Tone.Synth().toDestination();
            
                    // ボタンがクリックされたときに音を鳴らす
                    document.getElementById("playButton").addEventListener("click", () => {
                        synth.triggerAttackRelease("C4", "8n"); // C4の音を8分音符で鳴らす
                    });
                </script>
            </body>
            </html>

2.説明 

                1. Tone.jsのインクルード:
                   <script src="https://cdn.jsdelivr.net/npm/tone@14.8.28/build/Tone.js"></script> の部分で、Tone.jsをCDN経由で読み込みます。
                
                2. シンセサイザーの作成:
                   const synth = new Tone.Synth().toDestination(); で、シンセサイザーオブジェクトを作成し、
                   音の出力先（toDestination()）をデフォルトのオーディオ出力（スピーカー）に設定します。
                
                3. ボタンをクリックで音を鳴らす:
                   ボタンをクリックすると、synth.triggerAttackRelease("C4", "8n"); で、C4の音（中央のド）を8分音符の長さで鳴らします。
