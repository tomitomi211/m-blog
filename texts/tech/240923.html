うわさのzigを、macで動作手順

    # install
    $ brew install zig
    
    # 確認
    $ zig version

初期化：まだ書き方ゆれてるっぽい。参考にした記事の通りにやってもエラーになった。以下だとできた

    $ zig init
    $ zig build run

src/main.zigを以下に変更する

    const std = @import("std");
    
    pub fn main() !void {
    const stdout = std.io.getStdOut().writer();
    try stdout.print("Hello, {s}!\n", .{"world"});
    }

実行。これでhello worldまでできたとさ

    zig build run
