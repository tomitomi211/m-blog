Markdown Renderer

このページではmarkdownをCDNで簡単に使うためのテストを紹介!

基本的に以下のCDVをheadに埋め込みます!

        <script>src="https://cdnjs.cloudflare.com/ajax/libs/marked/2.1.3/marked.min.js"</script>

変換する場所も入れたければ以下がサンプルコードです！

        <textarea id="markdown" rows="10" cols="50" placeholder="ここにMarkdownを入力してください"></textarea>
            <button id="render">Render Markdown</button>
            
            <div class="markdown" id="result"></div>
            
            <script>
                document.getElementById('render').addEventListener('click', function() {
                    const markdownText = document.getElementById('markdown').value;
                    document.getElementById('result').innerHTML = marked(markdownText);
                });
            </script>
